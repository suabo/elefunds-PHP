elefunds PHP Library
====================

The elefunds PHP SDK to interact with the elefunds API via PHP.

## Getting started

Check out the included [documentation](https://github.com/elefunds/elefunds-PHP/tree/master/src/Elefunds/Documentation) and see the [SDK](https://github.com/elefunds/elefunds-SDK) repository for more information:

http://elefunds.github.io/elefunds-SDK/

### Installing via Composer

1. Install Composer:

        curl -s http://getcomposer.org/installer | php
    
2. Include `elefunds/sdk` in your project's `composer.json` file:

        {
            "require": {
                "elefunds/sdk": "1.2.*"
            }
        }
    
3. Install via Composer:

        php composer.phar install
